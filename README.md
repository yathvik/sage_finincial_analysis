# Sage Financial Analytics

A Python tool to fetch financial data from Sage Business Cloud Accounting (via CSV export) and generate income, expense, and profit reports.

## Features

- Loads sales invoices and purchase bills from CSV files
- Calculates total income, total expenses, and net profit
- Generates an Excel report with detailed breakdowns
- Creates a bar chart visualisation of financial summary

## Output

- `sage_report.xlsx` – Excel report with sales, purchases, and summary sheets
- `financial_summary.png` – Bar chart image
